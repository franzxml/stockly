# Stockly

## Deskripsi
Stockly merupakan sistem manajemen produk dan inventaris toko sederhana berbasis *web* yang dirancang untuk membantu usaha kecil dan menengah. Sistem ini memfasilitasi pengelolaan data produk, kategori, pemantauan stok masuk dan keluar, serta penyajian laporan inventaris secara terintegrasi dan efisien. Stockly bertujuan untuk menggantikan pencatatan manual yang rentan kesalahan dengan solusi digital yang mudah digunakan oleh pengguna nonteknis. Fitur utama mencakup manajemen akun admin, manajemen produk, manajemen kategori, manajemen stok, serta laporan produk dan inventaris.

## Cara Menjalankan
1. Unduh atau *clone* repositori ini ke komputer lokal Anda.
2. Buka folder proyek yang telah diunduh.
3. Jalankan berkas index.html menggunakan *browser* untuk mengakses halaman *login* aplikasi.
4. Setelah berhasil masuk atau untuk melihat tampilan dasbor utama, Anda dapat membuka berkas dashboard.html secara langsung di *browser*.
5. Jika ingin *live reload*, gunakan ekstensi Live Server di VS Code.
